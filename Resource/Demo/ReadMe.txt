Installation
Einfach das EXE starten.

Inhalt
Diese Anwendung demonstriert das prinzipielle Verhalten des Cocktailpro. Es handelt sich um eine alte Version und mittlerweile haben sich einige Anforderungen geändert! Viele Funktionen wurden nur teilweise implementiert! 

Sonstiges
Die Anwendung läuft im Demo-Modus, d.h. mit 10-facher Geschwindigkeit. Durch den Aufruf mit dem Parameter "normal" kann der Echtzeit-Modus aktiviert werden.

Systemanforderungen
Die Anwendung läuft nur unter Windows. Für Nicht-Windows-Benutzer gibt es einen Snapshot der Ausgabe unter Ausgabe.txt.

Enjoy
